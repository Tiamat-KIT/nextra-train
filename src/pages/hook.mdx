# Hookについて
Hook(フック)というのは、Reactにおいて、**状態**を扱うために用いられる関数です。
例えば、ボタンがクリックされた回数を記録しながら表示したいときは
`useState`というフック関数を使います。

```ts 
import {useState} from "react"
function Button(){
    const [count,setCount] = useState<number>(0)
    const clickEvent = () => {setCount(count += 1)}
    return (<button onClick={clickEvent}>Click</button>)
}
```
